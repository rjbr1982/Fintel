# 🧠 קונטקסט פרויקט (Project Context)

## 🎯 מטרת העל ("דוחכם" - Fintel)
אפליקציה חכמה לניהול תקציב משפחתי הכוללת קבועות, משתנות ועתידיות, מנוע לתכנון חירות פיננסית, מרכז חסכונות מנוהל (Sinking Funds), ומנוע סטטיסטיקת שכר דינמי.

## ⚙️ חוקים, הגדרות ומאפייני ליבה נוכחיים:
- **תשתית משפחה:** אחוזי מחיה ויעדים מחולקים אוטומטית לפי סטטוס זוגי וכמות ילדים. קיימת "קופה מאוחדת" להוצאות ילדים שמתחלקת שווה בשווה.
- **מנוע שכר דינמי:** הכנסות ממשכורת יכולות להינעל לעריכה ידנית ולשאוב את הממוצע החודשי שלהן מתוך "דשבורד הסטטיסטיקה" שמחשב שעות עבודה, ממוצע חודשי ו-YTD, כדי למנוע גירעון סמוי בתזרים.
- **מנוע מפל ושארית:** הכנסות פחות בסיס -> יתרה לרמת חיים וחירות. "קניות" מתפקד כעוגן קשיח במשתנות.
- **אפס אזהרות (Zero Warnings Policy):** חובה לשמור על Linter נקי בכל סוף סשן.

## 🏗️ סטייט וארכיטקטורה:
- UI מבוסס Flutter (תמיכה מלאה ב-Web).
- מסד נתונים: Firebase (Firestore, Auth).
- State Management: חבילת `provider`, כשהלוגיקה המרכזית יושבת ב-`BudgetProvider`. השאיפה היא UI אופטימי עם המתנה לפעולות השרת רק כשנדרש.
